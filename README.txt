Менеджер задач v1.0.0

Описание:
Простое и удобное приложение для управления задачами и списком дел.

Основные возможности:
- Добавление, редактирование и удаление задач
- Установка приоритетов задач (низкий, средний, высокий)
- Установка сроков выполнения задач
- Отметка задач как выполненных
- Фильтрация задач (все, активные, выполненные)
- Поиск по задачам
- Статистика выполнения задач
- Удаление выполненных задач

Использование:
1. Запустите файл "Менеджер задач.exe"
2. Используйте кнопку "Добавить задачу" для создания новой задачи
3. Двойной клик по задаче открывает окно редактирования
4. Используйте кнопки фильтров для просмотра разных категорий задач
5. Для отметки задачи выполненной нажмите "Пометить выполненной"

Структура данных:
- Задачи хранятся в локальной базе данных SQLite (tasks.db)
- Все данные сохраняются автоматически

Системные требования:
- Windows 7/8/10/11
- .NET Framework 4.5 или выше (обычно уже установлен)

Примечание:
Регулярно создавайте резервные копии файла tasks.db.
